# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Pledge

We are committed to maintaining a respectful, professional, and harassment-free environment for everyone participating in this project.

## Our Standards

Examples of behavior that contributes to a constructive environment include:
- Clear, technical, and respectful communication
- Focusing on reproducible issues and concrete improvements
- Respecting differing viewpoints and implementation approaches

Examples of unacceptable behavior include:
- Harassment, insults, or personal attacks
- Publishing private information without explicit consent
- Disruptive behavior that undermines productive collaboration

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, and other community interactions related to this repository.
